To access the API you need to use the following credentials:

User: my_user

You could test it via the following commands using curl. The API have different URL's depending on the format (JSON, XML)


Articles

JSON
curl -H 'Accept: application/json; indent=4' -u my_user:my_password http://198.211.108.179/services/articles/
XML
curl -H 'Accept: application/json; indent=4' -u my_user:my_password http://198.211.108.179/services_xml/articles/

Stores

JSON
curl -H 'Accept: application/json; indent=4' -u my_user:my_password http://198.211.108.179/services/stores/
XML
curl -H 'Accept: application/json; indent=4' -u my_user:my_password http://198.211.108.179/services_xml/stores/

Get Articles by Store

In this case you need to replace ':id' with a positive integer.

JSON
curl -H 'Accept: application/json; indent=4' -u my_user:my_password http://198.211.108.179/services/articles/stores/:id
XML
curl -H 'Accept: application/json; indent=4' -u my_user:my_password http://198.211.108.179/services_xml/articles/stores/:id {% endblock %}
